 # The Book Haven

### Live Site URL:  https://book-haven-48f66.web.app/
 
## About Book Haven
**Book Haven** is a modern web application where users can explore, add, and comment books easily.  
It’s built with React and Firebase authentication to provide a seamless reading community experience.


## Features

- **Secure Authentication:** Email-password and Google login using Firebase.
- **Book Management:** Add, view,delete and update book details dynamically.
- **Dark & Light Theme:** User-friendly theme toggle for personalized browsing.
- **User comments:** Readers can post and view book comments.
- **Dynamic User Profile:** Displays user image and name instantly after login.
- **Responsive Design:** Fully optimized for desktop, tablet, and mobile devices.

---

## Technologies Used

- **React.js** – Frontend Framework  
- **Firebase** – Authentication & Hosting  
- **MongoDB** – Database  
- **Express.js & Node.js** – Backend Server  
- **Tailwind CSS** – Styling  
- **React Router** – Navigation  
- **React Hot Toast / SweetAlert** – Notification System
